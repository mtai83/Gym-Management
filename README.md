# Cấu hình 
1. Chạy file gymmanagement.sql trong Database Management Tools ( MySQL trong Xampp) để tạo cơ sở dữ liệu
2. Chạy server backend:
  cd ...(thư mục code)..
  php -S localhost:5500

3. Mở trang web: http://localhost:5500/
